# ichiyaku

ボカロ「一龠」のMVの表現を再現（未完成）

## ローカルでの実行方法
1. グリフウィキにある全グリフデータ（http://glyphwiki.org/dump.tar.gz ）をダウンロードし、その中の`dump_newest_only.txt`を`./src/processingProjectFile/`に移動させる。

1. 以下のコマンドを実行し、プロジェクトファイルの前処理を行う。
    ```
    npm i
    npm run pj
    ```

1. 以下のコマンドを実行し、ローカルサーバーを立ち上げる。
    ```
    npm run dev
    ```
